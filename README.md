# JSONata Playground (Express.js)

A lightweight Node.js web app to run JSONata queries on your JSON data. Paste or load JSON, enter a JSONata query, and see the result instantly.

## Features

- Paste or upload JSON data
- Enter JSONata queries
- See results or errors in real time
- Minimal, fast, and easy to use

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the server:
   ```sh
   node index.js
   ```
3. Open your browser at [http://localhost:3000](http://localhost:3000)

## Project Structure

- `index.js` — Express.js backend and API
- `public/` — Frontend HTML, JS, and CSS

## Dependencies

- express
- jsonata
